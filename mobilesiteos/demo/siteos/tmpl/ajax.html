AJAX Filled Popup

This is a simple demo page showing how to fill our <div id="ajax_container">  popup with HTML text via an AJAX file retrieval. AJAX functionality is provided by jQuery. The HTML is injected, then an additional MobileSiteOSâ„¢ module is loaded and executed via a simple msos.require() module call. More complex JavaScript can be added by using multiple modules and additional functions through our msos.onload_functions.push() for queued arrays, followed by a call to msos.run_onload() to execute them after all msos.require() modules have fully loaded.
