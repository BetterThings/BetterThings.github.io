Mobile Grid Forms

This page demonstrates our responsive web design form elements. These form elements are loosely based on the excellent Twitter Bootstrap v2.3.2 Library of components, while being arranged by our version of responsive grid CSS code. Please note the use of the .form class as derived from Bootstrap v2.x.x. MobileSiteOS uses this simple CSS form structure as it's base by default, without the loading an additional module.
