MobileSiteOS™ JavaScript Demo

MobileSiteOS™ JavaScript is a highly modularized assortment of base JavaScript code. It adds the ability to load widgets and plug-ins for specific pages "on demand", or based on the page html code itself. Plus, MobileSiteOS™ controls the display size of pages based on the user's particular device or settings, while adding "responsive web design" capabilities.
