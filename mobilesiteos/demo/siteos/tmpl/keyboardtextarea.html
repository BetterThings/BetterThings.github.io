Keyboard CSS Positioning

Click the above button to display calculated keyboard element css positioning data in the above textarea. You can use this function to rebuild keyboard display css positioning. Start with an existing size css file and make changes to width and height css specifications as desired. Then rerun this function (for the correct size name) and copy/paste the new positioning data into that display size name css file. Don't forget: clear your browser's cache each time you reload the page.
