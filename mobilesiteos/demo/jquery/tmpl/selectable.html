jQuery-UI Selectable Widget

You can enable a DOM element (or group of elements) to be selectable with .selectable(). Draw a box with your cursor to select adjacent items. Hold down the Ctrl key to make multiple non-adjacent selections. We load the jQuery-UI .selectable() code seperately as a module, which in turn loads it's specific CSS file.
