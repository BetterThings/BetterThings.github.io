jQuery-UI Dialog Widget

The basic dialog window is an overlay positioned within the viewport, and is protected from page content (like <select> elements) shining through by an <iframe>. It has a title bar and a content area, plus can be moved, resized and closed with the '' icon by default.

This is the default dialog which is useful for displaying information. The dialog window can be moved, resized and closed with the '' icon.
