jQuery-Timeago

Timeago is a jQuery plugin that makes it easy to support automatically updating fuzzy timestamps (e.g. "4 minutes ago" or "about 1 day ago"). This plugin was written by Ryan McGeary.

You can also use it programmatically:

jQuery.timeago(new Date());
//=> ""

jQuery.timeago("2008-07-17");
//=> ""

jQuery.timeago(jQuery("abbr#some_id"));
//=> ""
//   [title="2008-07-20"]
