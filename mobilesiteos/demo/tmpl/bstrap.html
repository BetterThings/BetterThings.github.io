Bootstrap UI Example Pages

Here we have examples of Bootstrap UI integration into MobileSiteOS™  HTML5 web pages. The Bootstrap UI JavaScript code is nearly standard, with all widgets or specific code functions loaded as modules. Since our focus is "mobile first", we however have slightly massaged several of the Bootstrap UI CSS files on a case by case basis. This, in an effort to improve integration into MobileSiteOS™ and improve usability for mobile devices. There is however some stylesheet code which was derived from v2.3.2 (notably, the dropdown  and navbar code) which we still use as a base, but nearly the entirety of "separate module" code comes from v3.1.1. Needless to say, the transition from v2.3.2 to v3.1.1 was very painful and will be an ongoing process.
